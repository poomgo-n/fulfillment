품고 N배송 설정 가이드

[유의사항] N배송 설정 전 꼭 확인해주세요.

  • N배송 설정 완료 후 발생한 주문부터 품고에서 수집 및 출고됩니다.
  • 설정 이전 주문은 직접 출고가 필요할 수 있습니다.
  • 중복 수집이 발생하지 않도록 기존 플랫폼 연동 설정을 반드시 점검해주세요.
  • 중복 수집으로 인한 중복 출고는 품고에서 책임지지 않습니다.

1. 판매처 등록

경로 ) 품고 나우 > 주문 > 판매처 계정 > 판매처 계정 개별 등록

스마트스토어 판매처 계정을 등록하고, 연동 계정 정보를 정확하게 입력해주세요.

진행 방법

  1. 품고 나우 > 주문 > 판매처 계정 > 판매처 계정 개별 등록을 클릭합니다.
  2. 판매처명은 스마트스토어(물류사)로 설정합니다.
  3. 브랜드명은 조회되는 브랜드 중 해당 브랜드를 선택합니다.
  4. 연동 계정에는 스마트스토어 API 연동용 판매자 ID를 입력합니다.
확인 경로 : 스마트스토어 > 스토어 관리 > 서비스 연결 > API 관리 > API 연동용 판매자 ID

2. 풀필먼트 연동

경로 ) 스마트스토어 > 스토어 관리 > 네이버 서비스 연결 > 네이버 풀필먼트

N배송 사용을 위해 풀필먼트 계약 연동을 진행합니다.

진행 방법

  1. 풀필먼트 계약 연동을 클릭합니다.
  2. 물류사 목록에서 품고를 선택한 뒤 확인합니다.
  3. 정보 제공 동의 및 안내사항을 모두 확인 후 동의합니다.
동의 완료 후 익일 00:40 이후 자동으로 물류사 연동상태가 확인중에서 연동됨으로 변경됩니다.

3. N배송 프로그램 이용

경로 ) 스마트스토어 > N배송 관리 > N배송 프로그램 이용

풀필먼트 연동 완료 후 N배송 프로그램 이용 설정을 진행해주세요.

진행 방법

  1. N배송 프로그램 이용 화면에서 신청 가능한 상태인지 확인합니다.
  2. 필요한 조건이 모두 충족되었는지 점검합니다.
  3. 품고 안내에 따라 N배송 프로그램 이용 절차를 완료합니다.
판매처 등록, 풀필먼트 연동, 정보 동의가 모두 완료되어야 원활한 N배송 세팅이 가능합니다.
