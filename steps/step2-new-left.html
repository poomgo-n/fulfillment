1. 품고 나우 > 주문 > 판매처계정 > 판매처 계정 개별 등록을 클릭해주세요.
  2. ▪️판매처명 : 스마트스토어(물류사)
    ▪️브랜드명 : 노출되는 브랜드명 선택
    ▪️연동 계정 : API 연동용 판매자 ID 입력
    ▪️API 연동용 판매자 ID 확인 경로
    스마트스토어 > 스토어 관리 > 서비스 연결 > API 관리 > API 연동용 판매자 ID
  3. 모든 값을 선택 및 입력하였다면 등록을 클릭해주세요.

⚠️ 유의사항

  • API 연동용 판매자 ID을 붙혀넣는 과정에서 공백이나 다른 문자가 포함되지 않도록 유의해주세요.
  • 잘못된 계정 정보 입력 시 주문 수집 및 출고 연동이 정상적으로 진행되지 않을 수 있습니다.
  • 판매처 계정 등록 완료 후 반드시 저장 여부를 확인해주세요.
